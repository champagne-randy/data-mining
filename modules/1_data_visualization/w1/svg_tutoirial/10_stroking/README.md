## SVG Stroke Property

SVG offers a wide range of stroke properties. In this chapter we will look at the following:

* stroke
* stroke-width
* stroke-linecap
* stroke-dasharray

All the stroke properties can be applied to any kind of lines, text and outlines of elements like a circle.
